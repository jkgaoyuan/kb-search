#### 数据模型

```python
class DocumentVersion(SQLModel, table=True):
    id: int = Field(primary_key=True)
    document_id: int = Field(foreign_key="document.id", index=True)
    version_number: int  # 递增版本号
    title: str
    content: str  # 该版本的纯文本
    content_html: str
    change_summary: str | None  # 自动生成的变更摘要
    created_by: int = Field(foreign_key="user.id")

```

#### 技术要点

| 要点 | 方案 | 理由 |
|------|------|------|
| 存储策略 | 完整内容存储（非增量） | 回滚简单直接，磁盘成本可控（20版 x 1万文档=20万行，PostgreSQL轻松承载） |
| diff算法 | Python difflib.SequenceMatcher | 无需外部依赖，文本diff足够 |
| 变更摘要 | 自动提取前50字 | 人工补充可选，降低使用门槛 |
| 归档策略 | 保留最近20个版本 | 平衡恢复需求与存储成本，可配置 |

- [ ] 回滚操作生成新版本（保留回滚前状态）
- [ ] 超过20个旧版本自动清理

#### 风险与应对

| 风险 | 概率 | 预案 |
|------|------|------|
| 大文档版本存储膨胀 | 中 | 单文档版本限制5MB，超限时只存diff摘要 |

#### 推荐算法（简化版）

```python
def calculate_relevance(doc_a: Document, doc_b: Document) -> float:
    # 相关度 = 0.4 x 标签Jaccard + 0.3 x 分类邻近度 + 0.2 x 作者相同 + 0.1 x 时间邻近

    tag_score = jaccard_similarity(doc_a.tags, doc_b.tags)

    # 分类路径相似：同属"技术"大类 > 同属"技术/后端"子类
    cat_score = path_similarity(doc_a.category.path, doc_b.category.path)

    author_score = 1.0 if doc_a.author_id == doc_b.author_id else 0.0

    time_score = time_decay(doc_a.created_at, doc_b.created_at)

    return 0.4*tag_score + 0.3*cat_score + 0.2*author_score + 0.1*time_score
```

- [ ] 用户可标记"不感兴趣"，该文档不再推荐
- [ ] 推荐结果缓存1小时，更新文档后自动失效

```python
# app/services/auth.py
from abc import ABC, abstractmethod

class AuthProvider(ABC):
    @abstractmethod
    async def authenticate(self, credentials: dict) -> User:
        pass

    @abstractmethod
    async def get_user_info(self, token: str) -> dict:
        pass

class LocalAuthProvider(AuthProvider):
    # MVP阶段：JWT本地认证
    pass

class LDAPAuthProvider(AuthProvider):
    # 第三阶段：LDAP/AD认证
    pass

class OAuth2AuthProvider(AuthProvider):
    # 第三阶段：OAuth2认证
    pass

# 路由层根据配置动态选择
AUTH_PROVIDER = {
    "local": LocalAuthProvider(),
    "ldap": LDAPAuthProvider(),
    "oauth2": OAuth2AuthProvider(),
}[settings.AUTH_TYPE]
```

#### 环境变量配置

```bash
# 认证方式切换
AUTH_TYPE=local  # local | ldap | oauth2

# LDAP配置（AUTH_TYPE=ldap时生效）
LDAP_SERVER=ldap://corp.example.com
LDAP_BIND_DN=CN=admin,DC=example,DC=com
LDAP_BIND_PASSWORD=xxx
LDAP_USER_BASE_DN=OU=Users,DC=example,DC=com
LDAP_USER_FILTER=(sAMAccountName={username})

# OAuth2配置（AUTH_TYPE=oauth2时生效）
OAUTH2_CLIENT_ID=xxx
OAUTH2_CLIENT_SECRET=xxx
OAUTH2_AUTHORIZE_URL=https://login.example.com/oauth/authorize
```
